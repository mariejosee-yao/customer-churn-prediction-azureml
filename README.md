# Customer Churn Prediction using Azure Machine Learning

This project focuses on predicting customer churn using Machine Learning techniques in Microsoft Azure ML Studio.

## Technologies Used

* Python
* Scikit-learn
* Azure Machine Learning
* Pandas
* Power BI

## Project Objectives

* Predict customer churn behavior
* Perform data preprocessing and feature engineering
* Train and evaluate Machine Learning models
* Compare model performance

## Machine Learning Models

* Logistic Regression
* Random Forest

## Project Workflow

1. Data collection
2. Data preprocessing
3. Feature engineering
4. Model training
5. Model evaluation
6. Prediction and visualization
